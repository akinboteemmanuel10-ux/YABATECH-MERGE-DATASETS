## YABATECH-ADMISSION AND GRADUATION-MERGED-DATASETS

## Project Overview

This project focuses on cleaning, transforming, and analyzing YABATECH admission and graduation datasets using SQL.

## Objectives
- Clean inconsistent student records
- Remove duplicates
- Standardize date formats
- Merge datasets
- Generate useful insights
- Renaming columns and rows name

## Tools Used
- MySQL
- MySQL Workbench
- Excel( powerquery for merging datasets)
- GitHub

## Data Cleaning Steps
- Removed duplicates
- Corrected encoding issues
- Converted text dates to SQL dates
- Renamed columns
- Standardized student names

## Key SQL Concepts Used
- CTEs
- Window Functions
- Aggregate Functions
- ALter Table
- CASE Statements

## Insights
1. List top 10 students with highest Entry_Score per department. 
2. Calculate average Final_CGPA by department. 
3. Find number of students who graduated with Distinction. 
4. Show total admissions per year by Entry_Mode. 
5. Count students per state who had full scholarships. 
6. Retrieve all students who failed to graduate.
